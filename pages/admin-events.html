Neues Stadt-Event

Stadt-Events geben teilnehmenden Firmen für eine begrenzte Zeit Bonuspunkte. Beispiel: Bei einem Großbrand können Taxi, MD und PD zusätzliche Punkte erhalten.

Lege fest, welche Firmenarten vom Event profitieren. Wenn nichts ausgewählt ist, gilt das Event für alle.

Nach dieser Zeit endet das Event automatisch.
